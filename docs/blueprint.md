# **App Name**: StockPilot

## Core Features:

- Inventory Display: Display products in a modern card-based list with product photo, name, and stock count.
- Critical Stock Alert: Visually highlight products with stock levels below the critical threshold with a color change or icon.
- Detailed Product View: Display detailed stock information across different locations (Warehouse, Shop, Vehicle).
- Quick Stock Reduction: Provide prominent 'minus' buttons for quickly reducing stock at each location.
- Add New Product: A clean form design with fields for product name, critical threshold, initial stock levels (Warehouse, Shop, Vehicle), and an image selector.
- Order Generation & Sharing: Generate an order list and share using share sheet to different apps.
- Data Storage: Store product data, stock levels and images in Firestore and Storage

## Style Guidelines:

- Primary color: Deep Blue (#3F51B5) for a modern and professional feel.
- Background color: Light Gray (#F0F2F5), complementing the modern aesthetic, dark text should have enough contrast in Light Mode.
- Accent color: Cyan (#00BCD4) for highlighting interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern look.
- Use modern Card structures to list the products and follow Material Design 3 principles.
- Implement a modern Bottom Navigation Bar for main screen navigation.
- Employ soft animations to give a polished application feel and transitions.
